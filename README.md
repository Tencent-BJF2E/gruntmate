gruntmate
=========

一个Grunt可视化的软件

###GruntMate用途：
* 方便管理你基于Grunt的项目
* 方便统一管理Grunt插件
* 监听Gruntfile.js文件变化后，重启Grunt

###代码依赖于node-webkit来执行，需要到[node-webkit](https://github.com/rogerwang/node-webkit)下载

###需要有nodejs环境

##执行方式：
* 在src同级目录下：执行nw src
* 需要在src同级目录下，新建node_modules目录，将grunt需要用到的插件放在这里
* 程序左上角的+号是新建项目
